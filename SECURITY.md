# Security Policy

## Supported status

The published development target for this audit is `main`. GitHub currently reports `master` as the repository default branch, so maintainers should align that administrative setting. Security reports affecting either branch are accepted until the mismatch is resolved.

## Reporting a vulnerability

GitHub private vulnerability reporting is currently disabled. Open a minimal public issue asking the maintainer for a private contact channel; do not include exploit details, credentials, or sensitive evidence in that issue. If private reporting is enabled later, prefer GitHub's private reporting flow.

Include the affected route or file, reproduction steps, impact, and a suggested mitigation when available. Please allow reasonable time for review before public disclosure.

## Scope and limitations

ColdChain Sentinel is a synthetic-only, advisory-only demonstration. It has no user accounts, persistence, operational action tools, or real customer, shipment, pharmaceutical, patient, logistics, or sensor data. Deterministic rules remain authoritative, and optional AI output cannot authorize operational decisions.

Security reports are welcome, but this policy is not a certification or regulated-validation claim.
