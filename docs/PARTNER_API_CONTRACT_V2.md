# Partner API Contract v2

Phase 29 provides a static synthetic request/response contract, six-error catalog, samples, and OpenAPI 3.1 preview. Routes include `/partner-api-contract`, its JSON payload, OpenAPI preview, errors, and sample request/response JSON.

The inbound POST preview uses the Phase 2 sensor-reading fields. The outbound preview contains advisory evidence, deterministic authority, blocked actions, and human-review requirements. It is not a live partner API, enables no webhooks, requires no secrets or persistence, and makes no external calls.
