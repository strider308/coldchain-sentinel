# Demo Evidence Console

Phase 13 adds a single judge-facing console for the ColdChain Sentinel demo story.

Routes:

- `/demo-console`
- `/demo-console.json`
- `/judge-evidence`
- `/judge-evidence.json`

The JSON payload is synthetic-only, advisory-only, and dependency-light. It summarizes the command center, data model, trust pipeline, consensus, SERS, training lab, scenario lab, human review, replay, integration readiness, GPU research evidence, and Fireworks advisory explanation layer.

Safety boundary:

- deterministic rules remain authoritative
- Fireworks is optional explanation support
- GPU/Jupyter evidence is offline synthetic research evidence
- runtime boot requires no GPU or external service
- no operational action is automated

Manual Render deploy is still required after push.
