# Claims Boundary

ColdChain Sentinel is a synthetic hackathon/investor beta.

## Allowed Claims

- Synthetic data only.
- Deterministic rules remain authoritative.
- Fireworks is optional and non-authoritative.
- SERS and prediction outputs are advisory only.
- Benchmarks are on deterministic synthetic data against simple baselines.

## Disallowed Claims

- Production readiness.
- Medical, pharma, logistics, or compliance validation.
- Real-world release approval.
- Superiority over competitors.
- Autonomous release, quarantine, discard, reroute, or customer notification.
- AMD success.

Public dataset validation is not claimed. No external dataset is currently ingested, and future benchmark work remains isolated from production claims.
