# Public Dataset Adapters

ColdChain Sentinel currently uses synthetic data only. Candidate public dataset adapters are plans for mapping reviewed sources into Data Contract v2.

Before implementation, every source requires license/TOS, provenance, privacy-risk, schema, and data-quality review. Discovery directories are source-finding tools, not datasets. No external dataset is fetched or ingested by the current app.

Routes: `/dataset-adapters` and `/dataset-adapters.json`.
